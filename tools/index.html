A. Tools

Below are the tools that I think FTU CS students should know about.

Prerequisites: Curiosity, Coffe, Rainy Day + Lofi.

IDE & Visual Studio

Integrated Development Environments (IDEs) like Visual Studio Code provide essential features for coding: syntax highlighting, debugging tools, extensions, and intelligent code completion. These tools significantly enhance coding efficiency and project management.

Learn More →

Version Control & GitHub

Git and GitHub are fundamental for code version control and collaboration. They enable tracking code changes, managing different versions, collaborating with teams, and maintaining project history. Essential for both individual and team development.

Learn More →

API & Postman

APIs (Application Programming Interfaces) enable software communication. Postman is a popular tool for testing, documenting, and interacting with APIs. Essential for developing and testing web services and integrating different systems.

Learn More →

Cloud Platforms

Cloud platforms (AWS, GCP, Azure) provide scalable computing resources and services. They offer storage, computing power, ML services, and deployment tools. Essential for modern application deployment and scalable ML solutions.

Learn More →

arXiv & LaTeX

arXiv is a repository for research papers, while LaTeX is a document preparation system. Together, they're essential for academic writing, research documentation, and sharing scientific findings in a professional format.

Learn More →

UX/UI & Figma

User Experience (UX) and User Interface (UI) design are crucial for creating user-friendly applications. Figma is a collaborative design tool for creating interfaces, prototypes, and design systems with real-time team collaboration.

Learn More →

Tech Documentation

Technical documentation is crucial for project maintenance and knowledge sharing. Tools like Canva help create clear, visually appealing documentation for code, APIs, and project specifications.

Learn More →

Personal Portfolio

A professional online presence through personal websites and Google Scholar profiles showcases your work, research, and achievements. Essential for career development and networking in the tech industry.

Learn More →

AI Tools

Modern AI tools and prompt engineering skills help automate tasks and enhance productivity. Understanding how to effectively use AI assistants and integrate AI services is becoming essential in ML engineering.

Learn More →

Agile & Scrum

Agile methodologies and Scrum frameworks provide structured approaches to project management. These methods enable efficient team collaboration, iterative development, and continuous improvement in software projects.
